# RRAM

## Lanzar simulación

Para lanzar simulaciones primero se crean los archivos de configuracón empleando el archivo Init_simulation.py *cada vez que se ejecuta se elimina el contenido de la carpeta Init_data*. Una vez creados los archivos de configuración se ejecuta RRAM_Simulation.py que realizará las simulaciones, los resultados se guardan en la carpeta Results. *Cada vez que se ejecuta RRAM_Simulation.py se elimina el contenido de la carpeta Results*
